The inspiration for this project was the apparent demise of the SpringModules Project. SpringModules' Ehcache integration was used for a number of projects contributed to by the authors of Spring Ehcache annotations. When it was discovered that SpringModules no longer worked with version 3 of the Spring Framework, Spring Ehcache annotations was born.

The authors would like to give credit to the authors of SpringModules' Ehcache integration.

The @Cacheable annotation and it's application is largely modeled after the @Transactional annotation in the Spring Framework.

Dependencies

Spring Ehcache annotations is a Maven project. The primary source for the list of dependencies can be found in the Dependencies report.
